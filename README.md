## Coding style:  
1. Variable naming:
	- Snakecase for regulare variables:  
		`this_is_a_variable`
	- Pascalcase for class names:  
		`MyClass`
	- Camelcase for member functions:  
		`getType()`, `setType()`

2. Underscore before private variable
	`_a_private_variable`
